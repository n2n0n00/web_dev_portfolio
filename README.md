## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/restauranti.git
   cd into folder
   ```

2. Start the Development Server:
    ```bash
    npm start
    ```
